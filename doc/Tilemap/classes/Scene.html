Scene creation

Tilemap.Scene:createRandomLevel () Creates a random level with the size specified in the Tilemap Settings.
Tilemap.Scene:createEmptyLevel (levelWidth, levelHeight) creates an level filled with Tile Index 1

Render Methods

Tilemap.Scene:renderTiles () renders all tiles that are on screen, not the ones that are outside of the screen
Tilemap.Scene:renderObstacleTiles () renders all obstacle tiles (the engine calls this after drawing shadows
Tilemap.Scene:renderShading () renders lighted tiles with shadows and reflections
Tilemap.Scene:getObstacleMeshes () Returns a list of shadowcasters that will be created from obstacle tiles
Tilemap.Scene:renderCharacters () renders all characters from active scene

Tile coordinate calculations

Tilemap.Scene:getTileCoordinatesAtPosition (x, y) Returns tile coordinates under screen coordinates (camera won’t be recognized)
Tilemap.Scene:getTileCoordinatesByCamera (mx, my) Returns tile coordinates under screen coordinates with camera recognition
Tilemap.Scene:getTileCoordinatesUnderPlayer () Returns the tile coordinates under character with player index
Tilemap.Scene:getTileCoordinatesUnderCharacter (character) Returns tile coordinates under a character
Tilemap.Scene:getLevelWidth () Returns level width
Tilemap.Scene:getLevelHeight () Returns level height

Pathfinding

Tilemap.Scene:Route_getRoute (startX, startY, goalX, goalY) Returns the shortest path from start to goal as a list of tile coordinates
