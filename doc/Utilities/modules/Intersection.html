Utilities.Intersection.LineLine (p1x, p1y, v1x, v1y, p2x, p2y, v2x, v2y)
Calculates the intersection point of two lines line1 = p1 + r v1 line2 = p2 + s v2

Parameters:

  • p1x x-coordinate of p1
  • p1y y-coordinate of p1
  • v1x x-coordinate of v1
  • v1y y-coordinate of v1
  • p2x x-coordinate of p2
  • p2y y-coordinate of p2
  • v2x x-coordinate of v2
  • v2y y-coordinate of v2

Returns:

    if there is an intersection point: function returns {intersecX, intersecY} otherwise it returns nil
