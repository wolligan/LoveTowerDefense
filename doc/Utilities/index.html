Some useful Modules

Modules

init loads all utility modules
Tables Adds some more functions to the predefined lua table named “table”
Vector Implements several functions for 2D Vector operations
TextOutput Holds a string buffer and renders strings on screen.
Intersection Contains some helpful functions to check if and where two primitives collide
SlicedSprite Sliced Sprites let you draw sprites where the borders will only be stretched in one direction, corners won’t be stretched, center will be stretched in both directions.
Color Some helpful functions concerning Colors and some predefined colors
OO create classes and derived classes
